# Commands

```
/push @username <amount> <percent:optional> <title:optional>
/push @username <amount> <title:optional> <percent:optional>

Example:
/push @john 111+111 30% bread
/push @john 111+111 bread 30%
/push @john 111+111 bread
/push @john 111+111 30%
/push @john 111+111
/push @john 111
```
